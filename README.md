# MarkdownIntakeProcess
Shiny Application for Project Intake that knits markdown files in a notebook fashion.  This will have the ability to input project name and description.  It also has spots for outcomes and measures associated with the project.  While entering the application knits and renders a Markdown file for reproducible and tracking of the tasks.  It also includes an ace editor for the analyst to incorporate R code to make visuals.  Along with the editor a spreadsheet style table will be included for the users to enter track outside data sources when necessary.
